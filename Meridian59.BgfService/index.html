(3) Animated GIF

Query-URL

http://bgf.meridian59.de/render/{width}/{height}/{scale}/{file}/{anim}/{palette}/{angle}/?s={file};{anim};{palette};{hotspot}&s=...

Return

GIF File with Indexed Colors (up to 256 per frame). Image-Size is fixed, object might be only partially visible.

Parameters

{width} {height} {scale} {file} {anim} {palette} {angle} {hotspot}
Range/Syntax [16, 256] [16, 256] [1, 80] filename Cycle: Lo-Hi-T
None: Group		 [0, 255] [0, 7] [0,127]

(4) Palettes

Query-URL

http://bgf.meridian59.de/palette/{index}/{format}

Return

  • BMP: Image with 16x16 pixels and 256 Colors/Palette. Each palette entry appears once.
  • JSON: Array with 256 Integers (32-Bit ARGB colors with alpha set)

Parameters

{index} {format}
Range/Syntax [0, 255] bmp/json
